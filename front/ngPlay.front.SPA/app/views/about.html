Features

Echo

This was the first feature built and allowed for the creation of a thin vertical slice through all major layers.

When invoked a call is made to the WebAPI, which logs it to the database and returns the call.

Register / Login

This was the second feature implemented. It built upon the WebAPI Visual Studio project out-of-the-box functionality for registering and authenticating users. A custom ASP.NET Identity implementation was required to interface with PetaPoco.

Once a user has registered and logged in they are able to access restricted sections of the site.

Notes

This highly unimaginative feature was implemented third. Once a user has logged in they can Create, Retrieve, Update and Delete simple notes. The WebAPI controller restricts access to authenticated users.
